记一次黑苹果的安装

十一月 10, 2018
  • 黑苹果

首先贴下我的电脑配置

项目 描述
电脑型号 华硕 All Series 台式机
处理器 英特尔 第四代酷睿 i5-4590 @ 3.30GHz
主板 华硕B85M-F Plus （英特尔Haswell - Lynx Point）
内存 8GB（金士顿 DDR3 1600MHz）
主磁盘 台电 256GB A950（248GB/固态硬盘）
显卡 Nvidia GeForce GTX 750（2GB/七彩虹）
显示器 三星SAM0B65 S24D390（23.5英寸）
声卡 瑞昱 ALC887 @ 英特尔 Lynx Point 高保真音频
网卡 瑞昱 RTL8168/8111/8112 Gigabit Ethernet Controller / 华硕

准备工作

  1. 一台Mac，为了下载系统镜像
  2. 一个 > 8G的U盘

安装

  1. 使用mac电脑下载macOS High Sierra镜像，下载好后会存在/Applications目录下
  2. 插入U盘，使用Mac的磁盘工具把U盘格式化成Mac OS 扩展（日志式）
  3. 创建启动盘
    1. 下载 UniBeast（制作启动盘的工具） 工具并打开（需要英文系统才能使用，打开偏好设置–> 语言与地区–> 把首选语言改成英文，地址改成美国）
    2. 点击 Continue, Continue, Continue, Continue, Agree, 选择刚才的磁盘继续，选择UEFI Boot Mode继续，Continue，Install
    3. 大概10分钟左右完成写入
    4. 下载 MultiBeast（安装驱动的工具） 工具放入安装盘
  4. 启动PC按F2（不同主板进入BIOS的方式有所不同）进入BIOS
  5. 进入高级设置，修改硬盘为AHCI模式；修改XHCI为Enabled；修改系统为Other OS；保存BIOS设置，重启，选择EFI U盘启动，选择Boot OS X Install from Install macOS High Sierra
  6. 这时你会进入macOS安装界面，点击磁盘工具，格式化系统盘为Mac OS 扩展（日志式）
  7. 返回点击安装，选择刚才格式化的磁盘，等待重启安装完成
  8. 进入macOS系统，打开MultiBeast软件，安装相应的驱动，重启

问题

  1. 键盘 和 Mac的原生键盘不一样？
    • 点击系统偏好设置–> 键盘–> 设置键盘，win键 是 mac的Command键
  2. 无线网卡不能用？之前用的是腾达U9的无线网卡，在黑苹果上无法使用。
    • 购买小米随身WiFi (19.9买的)，安装小米随身WiFi驱动，重启就可以无线上网了
  3. 小米随身WiFi安装方法
    1. 下载驱动，密码 ky4q
    2. 下载并打开Kext_Utility工具，把RT2870USBWirelessDriver.kext文件拖到工具中
    3. 安装WirelessUtility，重启
    4. 重启后就能看到WiFi的图标了，打开连接wifi就可以愉快的上网了
