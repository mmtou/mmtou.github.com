maven

介绍

Apache Maven，是一个软件（特别是Java软件）项目管理及自动构建工具，由Apache软件基金会所提供。基于项目对象模型（缩写：POM）概念，Maven利用一个中央信息片断能管理一个项目的构建、报告和文档等步骤。wiki

Apache Maven is a software project management and comprehension tool.

安装

  1. 下载对应操作系统的maven安装包 并 解压

    ⚠️注意操作系统要求

  2. 配置环境变量

    • linux & mac: export PATH="$PATH:解压后的文件夹路径/bin
    • windows： 直接 计算机–>系统设置–>高级系统设置–>环境变量，编辑PATH，新增 ;解压后的文件夹路径/bin

  3. 运行mvn -v测试mvn是否安装成功

通常eclipse里已经集成了maven插件

pom.xml

pom.xml是maven项目的xml配置。
它包括项目基本配置、build配置、详细描述配置、环境配置等

https://maven.apache.org/pom.html
